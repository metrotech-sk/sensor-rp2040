Detailed Description

Analog input sensor class.

AnalogInput is a generic class for up to 4 analog inputs, e.g. 4x 0-3.3V

Note
default ADC depth on RP2040 is 12 bit, 8.7 ENOB (approx. 54dB SNR)
AnalogInput uses oversampling to increase resolution, increasing SNR by 6dB per bit
n-bit oversampling increases sampling time too: sample time = 4^n * conversion time

Parameters
numChannelsnumber of channels, e.g. 4
oversampleBitsnumber of bits to oversample, e.g. 3 bits -> 64x oversampling
Note
oversampleBits increase duration of update() by 4^oversampleBits, e.g. 3 bits -> 256 samples per channel per update
mean sampling time for 4 channels with no oversampling is 10us
mean sampling time for 2 channels with 3 bits oversampling is 5us * 4^2 = 320us
mean sampling time for 1 channel with 5 bits oversampling is 2.5us * 4^5 = 2560us
